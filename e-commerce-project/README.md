# Shopping Cart Web Application

A responsive Shopping Cart application built using HTML, CSS, JavaScript, Bootstrap, and Local Storage. Users can browse products, add items to the cart, manage quantities, calculate totals, and perform CRUD operations on products.

Live Link : https://e-commerce-122.netlify.app/

## Features

* Display Product Catalog
* Add Products to Cart
* Increase/Decrease Product Quantity
* Remove Products from Cart
* Automatic Cart Total Calculation
* Checkout Functionality
* Add New Products
* Update Existing Products
* Delete Products
* Bootstrap Modals
* Local Storage Integration
* Responsive UI

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Bootstrap 5
* Local Storage

## Functionality

### Product Management

* View all products
* Add new products
* Edit product details
* Delete products

### Cart Management

* Add products to cart
* Increase quantity
* Decrease quantity
* Remove items
* Calculate total amount
* Checkout and clear cart

### Data Persistence

* Products stored in Local Storage
* Cart data stored in Local Storage
* Data remains available after page refresh
