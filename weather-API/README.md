# 🌦️ Weather App

A simple and responsive **Weather Application** built using **HTML, CSS, and JavaScript** that fetches real-time weather data using the OpenWeatherMap API.


Live Link : https://weatherapi-101.netlify.app/

---

## 🚀 Features

* 🔍 Search weather by city name
* 🌡️ Displays temperature in Celsius
* 🌆 Shows city name
* 💧 Displays humidity
* 🌬️ Shows wind speed
* ⚠️ Handles invalid city input (Error message)
* 🎨 Beautiful UI with animations

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Async/Await)
* OpenWeatherMap API

## ⚙️ How It Works

1. User enters a city name
2. App sends request to OpenWeatherMap API
3. Fetches weather data
4. Displays:

   * Temperature
   * City Name
   * Humidity
   * Wind Speed

---

## 🔑 API Used

* OpenWeatherMap API
* Endpoint:

```
https://api.openweathermap.org/data/2.5/weather
```
